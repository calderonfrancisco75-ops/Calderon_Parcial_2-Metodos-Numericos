Este código pertenece a un parcial elaborado para la materia Métodos Numéricos, del segundo año de la Licenciatura en Física. Las consignas del mismo se encuentran en el pdf "consignas.pdf"
El archivo parcial2-p1.f90 corresponde a el programa madre correspondiente a la primera parte de las consignas (parcial2-p2.f90 a la segunda)
Sin embargo, como se utilizó la programación modular, la mayoría del código se encuentra en la carpeta "módulos"
El archivo "cmp.sh" es el compilador en shell para ejecutar tanto los programas de fortran como los gráficos de gnuplot en base a los resultados obtenidos.
